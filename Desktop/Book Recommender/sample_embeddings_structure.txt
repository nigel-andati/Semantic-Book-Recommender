# 📦 Precomputed Embeddings File Structure (7K Dataset Subset)
#
# Run create_embeddings_subset.py to generate from the original 7k books dataset:
#
# embeddings_7k_subset.zip
# ├── books_metadata.json    # 1,200 high-quality books with metadata
# └── embeddings.npy         # OpenAI embeddings (1200, 1536)
#
# ================================================================================
# 📄 books_metadata.json
# ================================================================================
#
# JSON array of book objects with metadata. Each book must have these fields:
# - title: string (book title)
# - author: string (author name, can be "Unknown")
# - genre: string (one of the supported genres)
# - mood: string (one of the supported moods)
# - description: string (detailed plot summary for search)
#
# Example structure:
# [
#   {
#     "title": "The Hobbit",
#     "author": "J.R.R. Tolkien",
#     "genre": "Fantasy",
#     "mood": "Adventurous",
#     "description": "A magical journey through Middle-earth with dwarves, dragons, and epic quests. Friendship, courage, and adventure in a fantastical world filled with treasure and danger."
#   },
#   {
#     "title": "Dune",
#     "author": "Frank Herbert",
#     "genre": "Sci-Fi",
#     "mood": "Adventurous",
#     "description": "A young noble navigates politics, religion, and ecology on the desert planet Arrakis. Space opera with complex world-building and intrigue."
#   }
# ]
#
# ================================================================================
# 🔢 embeddings.npy
# ================================================================================
#
# NumPy array saved with np.save() containing precomputed embeddings.
# Shape: (num_books, 1536) where 1536 is the OpenAI text-embedding-3-small dimension
# Each row corresponds to the book at the same index in books_metadata.json
#
# Generation code (run once with OpenAI API):
# ```python
# import openai
# import numpy as np
# import json
#
# # Load your books data
# with open('books_metadata.json', 'r') as f:
#     books = json.load(f)
#
# # Initialize OpenAI client
# client = openai.OpenAI(api_key="your-api-key")
#
# # Generate embeddings
# embeddings = []
# for book in books:
#     response = client.embeddings.create(
#         input=book['description'],
#         model="text-embedding-3-small"
#     )
#     embeddings.append(response.data[0].embedding)
#
# # Save as numpy array
# embeddings_array = np.array(embeddings)
# np.save('embeddings.npy', embeddings_array)
# ```

# 📤 Google Drive Upload Steps
# ================================================================================
#
# 1. Create zip file: `embeddings_package.zip` containing both files
# 2. Upload to Google Drive
# 3. Right-click file → "Get shareable link"
# 4. Change permissions to "Anyone with the link can view"
# 5. Copy the link and extract the file ID
# 6. Replace YOUR_GOOGLE_DRIVE_FILE_ID in the code with the file ID
#
# Example Google Drive URL format:
# https://drive.google.com/file/d/YOUR_FILE_ID/view?usp=sharing
